Um aplicativo onde os usuários podem adicionar, editar e remover tarefas, gerenciando suas atividades diárias de maneira prática e intuitiva.

Tecnologias usadas

HTML5 estrutura o conteúdo da web, CSS3 define o estilo e layout, e React cria interfaces dinâmicas e interativas usando componentes reutilizáveis.

HTML 5

É a versão mais recente da linguagem HTML, trazendo suporte nativo para multimídia, novos elementos semânticos e APIs para criar sites mais interativos e eficientes.

CSS 3

É a última versão das folhas de estilo em cascata, trazendo novos recursos como animações, transições, gradientes e layout flexível para aprimorar o design e a apresentação de páginas web.

React

É uma biblioteca JavaScript para criar interfaces de usuário, focada em componentes reutilizáveis e atualizações eficientes, ideal para construir aplicações web dinâmicas e interativas.
